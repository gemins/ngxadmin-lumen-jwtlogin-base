## Instalación y configuración

1. Ingresar a App y correr:
```sh
npm install
```
2. Ingresar a la carpeta Api y correr:
```sh
composer update
```
3. Correr el servicio en localhost:
```sh
php -S localhost:8000 -t public
```
4. Una vez iniciado el servidor del respositorio, ingresa a la ruta /key para copiar la clave de 32 chars y luego pegarlo en tu archivo .env (APP_KEY).
5. Rellenar tu archivo .env con los datos de tu base de datos.
5. Corre las migraciones y las semillas:

```sh
php artisan migrate
php artisan db:seed
```

4. Listo! Configura tu Lumen a gusto.
